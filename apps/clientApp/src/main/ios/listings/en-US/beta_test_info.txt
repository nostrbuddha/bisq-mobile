Thanks for testing Bisq Connect — a mobile client for the Bisq peer-to-peer network.

WHAT TO TEST:
• Pairing flow — pair with your Bisq2 desktop via QR code or pasted pairing code
• Offerbook and market data browsing from your trusted node
• User profile, payment accounts, and reputation views
• Tor connectivity (the app embeds a Tor client for .onion node addresses)
• Settings — language, trade rules

NO BISQ2 NODE? USE DEMO MODE:
Enter the pairing code BISQ_DEMO_PAIRING_CODE on the trusted-node setup screen to launch the app with simulated data. Demo mode runs fully offline and showcases the app's UI without a live node connection. Some features are intentionally limited in demo mode (offer creation, push notifications, live transaction flows).

REPORTING ISSUES:
Please report any bugs, crashes, or UX issues at github.com/bisq-network/bisq-mobile/issues with steps to reproduce, your iOS version, and the build number from Settings → About.
